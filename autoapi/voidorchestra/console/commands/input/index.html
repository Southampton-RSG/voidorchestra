voidorchestra.console.commands.input¶

This module contains sub-commands for void-orchestra add.

The commands should add new entries to the DB from a fixture file.

voidorchestra.console.commands.input.connect_to_database_engine(location: str) → sqlalchemy.engine.Engine¶

Create a connection to the database.

Return an SQLAlchemy Engine which creates a connection to the database. If the database does not exist, a new database is created. In the case of multiple calls of this function, a new engine will not be created and an already existing engine will be returned.

Parameters:

location (str) – The filepath to the database to open.

Returns:

engine – The engine object to connect to the database.

Return type:

sqlalchemy.engine

class voidorchestra.console.commands.input.SonificationProfile¶

Bases: voidorchestra.db.Base

ORM class for sonification profiles.

These are the patterns of sonification that are used to generate sonifications for data. They include an instrument, and how it is used.

id¶

A unique ID for the sonification profile.

Type:

integer

sonification_method_id¶

The ID of the sonification method used by this profile.

Type:

integer

name¶

The name of the sonification profile.

Type:

string

description¶

A description of the sonification profile.

Type:

string

tempo¶

The speed of the sonification.

Type:

float

key¶

The note(s) to use for the sonification.

Type:

text

sonification_method¶

The method used to sonify data as part of this profile. Can be synthesizer or instrument samples.

Type:

relationship

The sonifications generated using this profile.
